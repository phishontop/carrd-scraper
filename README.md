# Carrd Scraper 
One of the best carrd osint tools to automatically gather information and data on a certain carrd

### usage
```
git clone https://github.com/phishontop/carrd-scraper
cd carrd-scraper/
pip install requests
pip install bs4
python main.py
```

### Features
- Social Finder
- Gender Finder
- Is Under 18
- html scraper
- text scraper
